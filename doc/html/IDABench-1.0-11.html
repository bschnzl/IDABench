11. Additional notes

11.1 Debugging

The scripts fetchem.pl, cleanup.pl and pat_search.pl can be run from a commandline with several switches available to modify their actions. Two of them, fetchem.pl and cleanup.pl, accept a -debug switch which sends verbose debugging output to corresponding files in the /var/log directory. If a problem seems insurmountable, send that debugging output to the IDABench Developers list, idabench_dev@ists.dartmouth.edu, and we'll have a look.

11.2 Support

In addition to the documentation provided, ISTS has set up two public mailing lists where users and developers can ask questions on installation, configuration, performance and scalability, etc. To subscribe to either the IDABench Users list, idabench_users@ists.dartmouth.edu and IDABench Developers list, idabench_dev@ists.dartmouth.edu , send email with the word "subscribe" as the subject to:

idabench_users-request@ists.dartmouth.edu

or

idabench_dev-request@ists.dartmouth.edu

Happy hunting!
